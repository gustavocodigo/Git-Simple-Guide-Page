Git Commands

Git cheatseet

Git Basics

git init cria um novo repositorio git local

git clone é usado para fazer uma copia do repositorio

git add adicionar um arquivos para o proximo estagio de commit

git commit vai criar um print atual do projeto e salvar no diretorio git

Git Branching and Merging

git branch cria um novo branch no repositório Git local.

git checkout é usado para alternar entre branches ou restaurar arquivos de um branch específico.

git merge combina as alterações de um branch com outro, integrando as modificações em um novo commit.

git log exibe o histórico de commits do projeto, mostrando as informações relevantes, como autor, data e mensagem de commit.

Fazer mudanças

git status Listar arquivos novos ou modificados que ainda não foram comitados

git diff Listar mudanças e conflitos

git add [file] prepara o arquivo para commit

git commit -m "mesagem descritiva" confirmar todos os arquivos preparados para o histórico de versionamento

Sincronização

git remote add [name] [url] cria uma nova conexão com o reposirio remoto

git fetch obter todas as alterações para a mesma origem

git pull obter todas as alterações mais recentes para a origem então mesclar

git push é usado para subir modificaçoes do repositorio local para o repositorio remoto
